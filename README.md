## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files. 

## Configuring JSON Server

Run the below commands in the Terminal
npm install -g json-server
json-server --config json-server.json

## Making JSON Server Running
Run 'json-server --watch db.json' and Json Server will be running on 'http://localhost:3000/`. You can navigate to the URL and have an eye on the JSON Objects..
